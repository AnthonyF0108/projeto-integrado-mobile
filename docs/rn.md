# Regras de Negócio (RN)

- **RN01:** O campo RG é obrigatório para a finalização de qualquer transação financeira.

- **RN02:** O sistema de pagamento apenas deve ser desbloqueado para utilizadores autenticados com conta Google.

- **RN03:** A lista de favoritos é persistente e vinculada exclusivamente ao ID único do utilizador autenticado.

- **RN04:** O utilizador deve poder editar os dados do seu perfil (RG e CEP) a qualquer momento através da interface de perfil.

- **RN05:** A confirmação da compra depende obrigatoriamente do status de aprovação retornado pela API do Mercado Pago.
