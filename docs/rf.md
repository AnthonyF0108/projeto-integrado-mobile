--📌 Exemplos Gerais--

RF01 – O sistema deve permitir que o usuário realize cadastro informando nome, e-mail e senha.

RF02 – O sistema deve validar o login utilizando e-mail e senha.

RF03 – O sistema deve permitir a recuperação de senha via e-mail.

RF04 – O sistema deve permitir a atualização dos dados cadastrais do usuário.

RF05 – O sistema deve gerar relatórios em formato PDF.

RF06 – O sistema deve permitir a exclusão de conta mediante confirmação do usuário.

🛒 Exemplos – Sistema de E-commerce

RF01 – O cliente deve poder adicionar produtos ao carrinho de compras.

RF02 – O sistema deve calcular automaticamente o valor total da compra.

RF03 – O cliente deve poder escolher a forma de pagamento (cartão, PIX ou boleto).

RF04 – O sistema deve enviar um e-mail de confirmação após a finalização do pedido.

RF05 – O administrador deve poder cadastrar, editar e excluir produtos.

🎓 Exemplos – Sistema Acadêmico

RF01 – O aluno deve poder realizar matrícula em disciplinas disponíveis.

RF02 – O professor deve poder lançar notas no sistema.

RF03 – O sistema deve calcular automaticamente a média final do aluno.

RF04 – O aluno deve poder consultar seu histórico escolar.

RF05 – O sistema deve gerar boletim em formato PDF.

🏥 Exemplos – Sistema Hospitalar

RF01 – O sistema deve permitir o cadastro de pacientes.

RF02 – O médico deve poder registrar diagnósticos e prescrições.

RF03 – O sistema deve agendar consultas conforme disponibilidade.

RF04 – O sistema deve emitir relatórios de atendimentos realizados.
