# Requisitos Funcionais (RF)

- **RF01:** O sistema deve permitir a autenticação do utilizador através do Google Sign-In.
  
- **RF02:** O sistema deve permitir que o utilizador complete o seu perfil com o número do RG.

- **RF03:** O sistema deve integrar-se com uma API de localização para preenchimento automático de morada via CEP.
  
- **RF04:** O sistema deve permitir ao utilizador marcar e desmarcar produtos como "Favoritos".
  
- **RF05:** O sistema deve processar pagamentos de forma funcional para concluir a compra.
  
- **RF06:** O sistema deve exibir um catálogo de produtos organizados de forma intuitiva.
  
- **RF07:** O sistema deve manter um histórico das encomendas finalizadas pelo utilizador.
