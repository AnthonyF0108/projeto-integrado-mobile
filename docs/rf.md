# Requisitos Funcionais (RF)

Os requisitos funcionais descrevem as funcionalidades que o sistema deve possuir.

RF01 – O sistema deve permitir o cadastro de produtos.

RF02 – O sistema deve permitir visualizar os produtos cadastrados.

RF03 – O sistema deve permitir registrar vendas.

RF04 – O sistema deve permitir visualizar o histórico de vendas.

RF05 – O sistema deve permitir editar produtos cadastrados.

RF06 – O sistema deve permitir excluir produtos.

RF07 – O sistema deve atualizar o estoque automaticamente após cada venda.

RF08 – O sistema deve permitir o cancelamento (estorno) de vendas realizadas.

RF09 – O sistema deve permitir a autenticação de usuários (login) para acesso ao sistema.

RF10 – O sistema deve gerar resumos financeiros e relatórios de vendas.
