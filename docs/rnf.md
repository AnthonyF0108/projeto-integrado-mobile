📌 Exemplos Gerais

RNF01 – Desempenho: O sistema deve responder às requisições do usuário em até 3 segundos.

RNF02 – Disponibilidade: O sistema deve estar disponível 99% do tempo durante o horário comercial.

RNF03 – Segurança: As senhas devem ser armazenadas de forma criptografada.

RNF04 – Usabilidade: O sistema deve possuir interface responsiva para dispositivos móveis.

RNF05 – Confiabilidade: O sistema deve realizar backup automático diário dos dados.

RNF06 – Compatibilidade: O sistema deve funcionar nos navegadores Chrome, Firefox e Edge.

RNF07 – Escalabilidade: O sistema deve suportar até 5.000 usuários simultâneos.

RNF08 – Manutenibilidade: O código deve seguir padrões de desenvolvimento documentados.

🛒 Exemplo – Sistema de E-commerce

RNF01 – Desempenho: O cálculo do valor total da compra deve ocorrer em menos de 1 segundo.

RNF02 – Segurança: O sistema deve utilizar protocolo HTTPS para transações financeiras.

RNF03 – Disponibilidade: O sistema deve estar disponível 24 horas por dia, 7 dias por semana.

RNF04 – Escalabilidade: O sistema deve suportar picos de acesso em períodos promocionais.

🎓 Exemplo – Sistema Acadêmico

RNF01 – Acessibilidade: O sistema deve seguir diretrizes de acessibilidade (ex: contraste adequado e navegação por teclado).

RNF02 – Desempenho: A consulta ao histórico escolar deve carregar em até 2 segundos.

RNF03 – Segurança: O acesso às notas deve exigir autenticação do usuário.
