# Requisitos Não Funcionais (RNF)

- **RNF01:** A aplicação deve ser desenvolvida em ambiente Flutter para garantir performance nativa.

- **RNF02:** A autenticação e persistência de sessão devem ser geridas via Firebase/Google Auth.
  
- **RNF03:** A interface deve seguir os princípios de UX para dispositivos móveis.
  
- **RNF04:** A aplicação deve ser capaz de lidar com requisições externas de forma assíncrona.
