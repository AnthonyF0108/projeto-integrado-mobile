# Requisitos Não Funcionais (RNF)

RNF01 – O sistema deve possuir interface simples e intuitiva.

RNF02 – O sistema deve funcionar em dispositivos Android.

RNF03 – O sistema deve garantir a segurança e integridade dos dados através de criptografia simples.

RNF04 – O sistema deve apresentar bom desempenho, processando vendas em menos de 2 segundos.

RNF05 – O sistema deve ser fácil de usar para usuários iniciantes (Usabilidade).

RNF06 – O sistema deve persistir os dados localmente ou em nuvem para evitar perda de informações.

RNF07 – A interface deve ser responsiva para diferentes tamanhos de tela de smartphones.
