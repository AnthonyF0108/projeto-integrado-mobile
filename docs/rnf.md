# Requisitos Não Funcionais (RNF)

Os requisitos não funcionais descrevem características de qualidade do sistema.

RNF01 – O sistema deve possuir interface simples e intuitiva.

RNF02 – O sistema deve funcionar em dispositivos Android.

RNF03 – O sistema deve garantir a segurança e integridade dos dados.

RNF04 – O sistema deve apresentar bom desempenho durante o registro de vendas.

RNF05 – O sistema deve ser fácil de utilizar para usuários iniciantes.
