# Backlog do Produto

O backlog do produto representa a lista de funcionalidades que devem ser desenvolvidas no sistema AgroVale, organizadas em épicos e user stories.

---

## Épico 1 – Autenticação e Cadastro de Usuário

| User Story | Descrição | Prioridade |
|------------|-----------|------------|
| US01 | Como usuário, quero fazer login com minha conta Google para acessar o aplicativo rapidamente sem criar uma senha | Alta |
| US02 | Como usuário, quero me cadastrar com e-mail e senha para ter uma conta própria no sistema | Alta |
| US03 | Como usuário, quero fazer login com e-mail e senha para acessar minha conta | Alta |
| US04 | Como usuário autenticado, quero preencher meus dados pessoais (CPF, RG, telefone) para completar meu cadastro | Alta |
| US05 | Como usuário autenticado, quero informar meu CEP e ter meu endereço preenchido automaticamente para agilizar o cadastro | Alta |
| US06 | Como usuário autenticado, quero encerrar minha sessão para proteger minha conta em dispositivos compartilhados | Alta |

## Épico 2 – Catálogo de Produtos

| User Story | Descrição | Prioridade |
|------------|-----------|------------|
| US07 | Como usuário, quero visualizar todos os produtos disponíveis em uma grade para conhecer o catálogo da loja | Alta |
| US08 | Como usuário, quero buscar produtos pelo nome ou categoria para encontrar o que preciso rapidamente | Alta |
| US09 | Como usuário, quero visualizar os detalhes de um produto (imagem, preço e descrição) para decidir se desejo comprá-lo | Alta |

| US11 | Como usuário autenticado, quero visualizar minha lista de favoritos para acessar rapidamente os produtos salvos | Média |

## Épico 3 – Carrinho de Compras

| User Story | Descrição | Prioridade |
|------------|-----------|------------|
| US12 | Como usuário autenticado, quero adicionar produtos ao carrinho para reunir os itens que desejo comprar | Alta |

## Épico 4 – Pagamentos

| User Story | Descrição | Prioridade |

| US16 | Como usuário autenticado, quero pagar meu pedido via Pix para realizar o pagamento de forma rápida e segura | Alta |

## Épico 6 – Assistente Virtual com IA

| User Story | Descrição | Prioridade |
|------------|-----------|------------|
| US24 | Como usuário, quero descrever minha necessidade ou dificuldade para receber uma recomendação personalizada de produto | Alta |
| US25 | Como usuário, quero visualizar os produtos recomendados pela IA em cards com foto e preço para facilitar minha decisão de compra | Alta |
